# PythonVerse 🐍

Curso interativo de Python do zero ao desenvolvimento web.

## Deploy no Render.com (gratuito)

1. Crie conta em https://render.com
2. New → Web Service → conecte este repositório GitHub
3. Em **Environment Variables**, adicione:
   - Key: `ANTHROPIC_API_KEY`
   - Value: sua chave em https://console.anthropic.com/keys
4. Clique em **Deploy**

Pronto — o curso fica online com a Pyra IA funcionando!

## Rodar local

```bash
ANTHROPIC_API_KEY=sua-chave node server.js
```

Acesse: http://localhost:3000
